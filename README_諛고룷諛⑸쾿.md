# 터블/키친플래그 B2B 월정산 자동화 웹앱

이 폴더는 Streamlit Cloud에 그대로 올려서 링크형 웹앱으로 배포할 수 있는 패키지입니다.

## 핵심 기능

- RAW 출고리스트 엑셀 업로드
- 표준 상품/공급가 관리
- 업체별 예외 공급가 관리
- 거래처별 정산 제외 여부 관리
- 같은 업체 + 같은 주문일 + 같은 수취인 + 같은 주소 기준 배송비 1회 3,000원 계산
- 동일 배송묶음 내 카톤수량 이상 출고 시 배송비 무료 처리
- 업체별 거래내역서 엑셀 파일 ZIP 다운로드
- 상품 미매칭 / 단가 누락 / 수량 오류 확인
- 설정 백업/복원 파일 다운로드 및 업로드

## 파일 구성

- `app.py` : 웹앱 본체
- `requirements.txt` : Streamlit Cloud 설치 패키지 목록
- `.streamlit/config.toml` : 앱 기본 설정
- `seed_data/product_master.csv` : 초기 표준 단가 데이터

## Streamlit Cloud 배포 순서

1. GitHub에 새 Repository를 만듭니다.
2. 이 폴더 안의 파일 전체를 GitHub Repository에 업로드합니다.
3. https://streamlit.io/cloud 에 접속합니다.
4. `Create app` 또는 `New app`을 누릅니다.
5. GitHub Repository를 선택합니다.
6. Main file path를 `app.py`로 지정합니다.
7. 앱 주소를 원하는 이름으로 지정합니다. 예: `tubble-settlement`
8. Deploy를 누릅니다.
9. 배포가 끝나면 `https://tubble-settlement.streamlit.app` 같은 링크로 접속합니다.

## 사용 순서

1. `상품/표준 공급가` 메뉴에서 표준 단가를 확인합니다.
2. 네고단가가 있는 경우 `업체별 예외 공급가` 메뉴에 거래처명/상품명/예외공급가를 입력합니다.
3. `정산파일 생성` 메뉴에서 RAW 출고리스트를 업로드합니다.
4. 오류 데이터가 있으면 상품명 또는 공급가를 보완합니다.
5. `업체별 거래내역서 ZIP 다운로드` 버튼으로 업체별 엑셀파일을 받습니다.
6. 단가/거래처를 수정한 뒤에는 `설정 백업/복원` 메뉴에서 설정 파일을 다운로드해 보관합니다.

## RAW 출고리스트 필수 컬럼

- 배송사
- 송장번호
- 파트너사
- 주문일
- 주문 상품명
- 수량
- 수취인
- 수취인 주소

## 중요 주의사항

Streamlit Cloud 무료 배포 환경에서는 서버가 재시작되거나 앱이 재배포되면 앱 내부에 저장한 DB 파일이 초기화될 수 있습니다. 
따라서 실제 업무에서는 단가/거래처를 수정한 후 반드시 `설정 백업/복원` 메뉴에서 `b2b_settlement_settings.xlsx` 파일을 다운로드해 보관하세요.

개인정보가 포함된 출고리스트를 외부 클라우드에 업로드하게 되므로, 회사 내부 보안정책 확인 후 사용하세요. 보안이 중요하면 사내 서버 배포 방식이 더 적합합니다.

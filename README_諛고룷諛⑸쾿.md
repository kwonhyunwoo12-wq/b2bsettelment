# 터블/키친플래그 B2B 월정산 자동화 웹앱

## 주요 기능
- RAW 출고리스트 엑셀 업로드
- 상품명 유사매칭 자동 적용
- 파트너사 오타/표기차이 자동 묶음
- 신규 거래처 RAW 업로드 시 자동 추가
- 표준 공급가 관리
- 업체별 예외 공급가 관리
- 배송비 자동 계산: 같은 파트너사 + 같은 주문일 + 같은 수취인 + 같은 주소 기준 1회 3,000원
- 카톤수량 이상 무료배송 처리
- 거래처별 거래내역서 엑셀 ZIP 다운로드
- 설정 백업/복원

## Streamlit Cloud 배포 방법
1. GitHub 저장소에 이 폴더 안의 파일 전체를 업로드합니다.
2. Streamlit Cloud에서 New app 또는 Create app을 클릭합니다.
3. Repository는 GitHub 저장소를 선택합니다.
4. Branch는 main을 선택합니다.
5. Main file path는 `app.py`로 입력합니다.
6. Deploy를 클릭합니다.

## GitHub 업데이트 방법
기존 저장소를 수정할 때는 ZIP 파일 자체를 올리지 말고, 압축을 푼 뒤 아래 파일/폴더를 GitHub에 업로드하세요.

- app.py
- requirements.txt
- seed_data 폴더
- .streamlit 폴더
- README_배포방법.md

기존 파일과 같은 이름으로 업로드하면 덮어쓰기/교체됩니다. 업로드 후 `Commit changes`를 누르면 Streamlit Cloud가 자동으로 재배포합니다.

## 사용 순서
1. 상품/표준 공급가 메뉴에서 표준단가를 확인하거나 수정합니다.
2. 업체별 예외 공급가 메뉴에서 네고단가가 있는 거래처만 입력합니다.
3. 정산파일 생성 메뉴에서 RAW 출고리스트를 업로드합니다.
4. 상품명 매칭 현황과 파트너사 자동묶음 현황을 확인합니다.
5. 오류가 있으면 상품/거래처/단가를 수정한 뒤 다시 업로드합니다.
6. 업체별 거래내역서 ZIP을 다운로드합니다.

## 주의사항
Streamlit Cloud 무료 환경은 앱이 재시작되면 내부 저장 DB가 초기화될 수 있습니다. 단가/거래처 정보를 수정한 뒤에는 반드시 설정 백업/복원 메뉴에서 백업 엑셀을 다운로드해두세요.

## v4 추가 반영사항: 세트/수량형 상품명 자동 계산

상품마스터가 단품 기준이어도 RAW 상품명에 아래 형태가 들어오면 공급가를 자동 계산합니다.

- `포밍 주방세제 본품+리필` → `포밍 주방세제 본품` 단가 + `포밍 주방세제 리필` 단가
- `세정티슈 * 10개` → `세정티슈` 단가 × 10
- `세정티슈 10개 세트` → `세정티슈` 단가 × 10

정산 결과 미리보기에서 `공급가계산방식`, `세트구성` 컬럼으로 어떤 단품을 기준으로 계산했는지 확인할 수 있습니다.
거래처 제출용 엑셀에는 기존과 동일하게 `공급가`, `상품금액`, `총액`이 반영되어 출력됩니다.

## v5 추가 반영사항: 실제 RAW 출고리스트 직접 처리

기존에 사용자가 수기로 정리하던 가공본뿐 아니라, 창고/택배사에서 받은 실제 RAW 출고리스트도 직접 업로드할 수 있습니다.

실제 RAW 기준 주요 처리 방식:

- `주문번호2`를 정산용 `파트너사`로 사용합니다.
- `주문 상품명`을 거래처 제출용 상품명으로 사용합니다.
- `자사 상품명`은 거래처 제출용으로 출력하지 않고, 세트/구성품 단가 계산용 보조근거로만 사용합니다.
- 동일 주문/송장/수취인/주소/주문상품명인데 자사상품명 기준으로 여러 줄로 쪼개진 경우, 주문상품명 기준 1건으로 자동 병합합니다.
- 병합된 행의 공급가는 RAW의 자사상품명 구성과 출고량을 기준으로 단품 공급가를 합산하여 계산합니다.

예시:

- RAW 2줄
  - 주문상품명: `터블 이지클린 텀블러 발포 세정제 본품(30정) X 2개 + 실속팩(100정) X 1개`
  - 자사상품명 1: `본품(30정)`, 출고량 2
  - 자사상품명 2: `실속팩(100정)`, 출고량 1
- 앱 처리 결과 1줄
  - 주문상품명은 그대로 유지
  - 공급가 = 본품 단가 × 2 + 실속팩 단가 × 1
  - 거래처 제출용 파일에는 주문상품명 기준으로 1건만 출력

검증용으로 앱 미리보기/전체거래내역에는 `자사상품명_수량구성`, `RAW병합라인수`, `공급가계산방식`, `세트구성` 컬럼이 표시됩니다. 거래처별 개별 정산서에는 기존처럼 거래처 제출에 필요한 기본 컬럼만 출력됩니다.
